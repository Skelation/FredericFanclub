# Frederic Fanclub

Static site for the **Frederic Fanclub** Valorant team: home, roster, about, and match history.

## Run locally

```bash
npm install
npm start
```

Open **http://localhost:3000** (serves the project root). You can also open `index.html` directly in a browser; using `npm start` matches how the site is usually deployed behind a web server.

## MVP contents

- `index.html` — Home
- `matches.html` — Match list (edit the markup to update results)
- `images/logo.svg` — Team mark (replace with your own art anytime)
- Roster cards use initials on gradient panels until you add jersey images again
